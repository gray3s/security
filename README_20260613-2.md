## Subfolders

- wireless/
  - Wireless network security: Wi-Fi, Bluetooth, hotspot/tethering, cellular-adjacent network issues, nearby-device discovery, Android wireless sharing behavior.
- wired/
  - Ethernet, USB tethering, cables, routers, switches, physical network interfaces.
- socialmedia/
  - Social media account security, platform exposure, privacy, impersonation, metadata, and public-posting risk.
- physical/
  - Physical access security, device custody, ports, locks, tamper exposure, and offline attack surfaces.
